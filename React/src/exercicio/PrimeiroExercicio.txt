# Exercício 4 - Gerador de Senhas (1)

Crie uma nova aplicação React com Vite que seja capaz de gerar senhas aleatórias. A senha deve ser exibida na tela juntamente com um botão de “Copiar” que copia o texto da senha para a área de transferência. Após copiar,
o botão para a área de transferência deve mudar seu texto para
“Copiado!” e, após gerar uma nova senha, o texto deve voltar 
para “Copiar”.

Imagens de referência:


            Gerador de senhas

             Gerar    Copiar

                  senha
