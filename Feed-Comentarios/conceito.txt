# Exercício 5 - Feed de comentários (1)

Crie um componente React de área de comentários onde deverá ser possível preencher e ler
comentários como nas imagens abaixo.
O formulário de preenchimento deverá ter um campo para o email e outro para o comentário e,
após ser submetido, deverá salvar o comentário do usuário em um estado e exibir o seu conteúdo 
na tela logo abaixo. Os comentários deverão estar ordernados do mais recente para o mais antigo.

Além disso, se não houver nenhum comentário uma mensagem deverá ser exibida como na imagem abaixo.

Seção de comentarios

Email  -> input

Comentario -> input

Enviar comentario -> botao de submissão
